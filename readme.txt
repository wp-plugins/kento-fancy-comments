Comment Author Profile on hover box.

== Description ==

Popup hover box for comment author profile for users and guest commenter.

Live Preview: http://kentothemes.com/demo/kento-fancy-comments/kento-fancy-comments/

Setting Details:
http://kentothemes.com/demo/kento-fancy-comments/kento-fancy-comments/



Plugin Features

* Any themes support.
* Custom hover box top position from mouse.
* User bio description from user profile.
* Author profile link or guest link.
* Random background color for comment.





== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress dashboard and activate it.
3. Please go plugin setting page by clicking "KFC Settings" on dhashboard left menu
4. Set a exact value to fix position for Popup box top position on hover.
5. And then go your comment list on single post see the CSS class (usualy comment-list or commentlist) ol(HTML) list used on your themes. and set this value to "Select class of comment-list" box on admin settings


<strong>How to get CSS class ?</strong>


To get class of themes comment list go your single post page and right click on mouse "view page source(html)" and find

<ol class="comment-list"> or
<ol class="commentlist">

<strong>How to fix hover box position with mouse ?</strong>
Go plugin admin setting and set a value(could be negative or positive) for "Popup box top position on hover" until popupbox reached closed to mouse position.
